# Inventory Management System

## Quick Installation

    git clone https://github.com/ekramasif/SD_Project.git

    cd SD_Project

    composer install
    
 ### For Database connect copy all content from ```.env.example```, create a ```.env``` file & paste here!

    php artisan migrate
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
